## 3. 使用示例

### 场景一：智能 3D 打印（结合数字孪生和知识库）

1.  **您说**："用 PLA 材料打印一个花瓶，温度设置为 250°C"

2.  **系统响应**：
    - **[数字孪生]**：在 51World 中模拟打印过程，发现温度过高。
    - **[知识库]**：检索到“PLA 材料的最佳打印温度是 220°C”。
    - **[自主学习]**：分析历史数据，发现您之前使用 220°C 打印成功率最高。
    - **[系统]**："警告：250°C 对于 PLA 材料过高。根据知识库和您的历史数据，建议将温度调整为 220°C。是否继续？"

3.  **您说**："好的，用 220°C 打印"

4.  **系统响应**：
    - **[拓竹数据对接]**：开始打印，并实时显示温度、进度和剩余时间。

### 场景二：智能无人机飞行（结合数字孪生和自主学习）

1.  **您说**："让无人机飞到我家周围，拍摄全景照片"

2.  **系统响应**：
    - **[自主学习]**：分析历史数据，发现您通常在 50 米高度拍摄全景。
    - **[数字孪生]**：在 51World 中模拟飞行路径，发现附近有高楼，50 米高度可能存在风险。
    - **[系统]**："根据您的习惯，建议飞行高度为 50 米。但在数字孪生中模拟发现附近有高楼，建议将高度提升至 80 米以确保安全。是否继续？"

3.  **您说**："好的，飞到 80 米"

4.  **系统响应**：
    - **[无人机控制]**：无人机起飞，飞到 80 米高度，执行拍照任务。

## 4. 如何部署和使用

### A. 部署

1.  **从您的 GitHub 仓库拉取最新代码**。
2.  **运行融合脚本** `MERGE_ENHANCEMENTS_TO_UFO.bat`。
3.  **运行一键启动脚本** `INSTALL_AND_START.bat`。

### B. 使用

- **知识库**：系统会自动将您的操作记录和关键信息存入知识库。
- **自主学习**：系统会在后台自动分析您的使用模式，并优化参数。
- **数字孪生**：在执行高风险任务（如无人机飞行、3D 打印）前，系统会自动进行模拟和预演。
