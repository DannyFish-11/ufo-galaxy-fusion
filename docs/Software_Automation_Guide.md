## 2. 工作原理

当您通过 Android 悬浮窗或 Windows 侧边栏发出一个命令时，系统会按以下流程工作：

```mermaid
graph TD
    A[用户输入: "把手机上最新照片发到电脑微信文件传输助手"] --> B(Node 50: NLU 引擎);
    B --> C{意图识别: 跨设备任务};
    C --> D[任务规划];
    D --> E[步骤 1: 在 Android 获取最新照片];
    D --> F[步骤 2: 在 Android 发送到电脑];
    D --> G[步骤 3: 在 Windows 打开微信];
    D --> H[步骤 4: 在 Windows 发送文件];
    
    subgraph 任务编排器
        E --> I(Android Agent 执行);
        F --> I;
        G --> J(Windows Agent 执行);
        H --> J;
    end
    
    I --> K[完成];
    J --> K[完成];
```

## 3. 使用示例

现在，您可以尝试以下这些强大的自然语言命令：

### 场景一：Windows 软件操作

- **“打开记事本，然后输入‘你好，世界！’”**
  - **系统动作**: 启动 `notepad.exe` -> 自动输入文本。

- **“在浏览器里搜索‘什么是量子计算’”**
  - **系统动作**: 打开 Chrome/Edge -> 在地址栏输入并搜索。

- **“打开微信，给‘文件传输助手’发送一张猫的图片”**
  - **系统动作**: 启动微信 -> 搜索“文件传输助手” -> 打开聊天窗口 -> 拖入图片并发送。

### 场景二：Android App 控制

- **“打开手机设置，然后进入 WLAN 页面”**
  - **系统动作**: 启动设置 App -> 查找并点击“WLAN”选项。

- **“在手机上打开微信，然后发送一条朋友圈说‘今天天气真好’”**
  - **系统动作**: 启动微信 -> 点击“发现” -> 点击“朋友圈” -> 点击右上角相机图标 -> 输入文字并发送。

### 场景三：跨设备协同

- **“把手机上的最新一张截图发送到我的电脑桌面”**
  - **系统动作**: **Android Agent** 找到最新截图 -> **任务编排器** 协调传输 -> **Windows Agent** 接收并保存到桌面。

- **“在电脑上生成一个‘日落’的视频，然后发送到我的手机”**
  - **系统动作**: **Windows Agent (Node 48)** 生成视频 -> **任务编排器** 协调传输 -> **Android Agent** 接收并保存到相册。

## 4. 如何部署和使用

所有增强功能都已集成到您 GitHub 仓库的 `enhancements` 目录中，并已包含在一键融合脚本 `MERGE_ENHANCEMENTS_TO_UFO.bat` 里。

1.  **确保您已运行过融合脚本**，将我们的增强功能集成到您 E 盘的微软 UFO 项目中。
2.  **在 Android 手机上安装最新的 APK**，并确保开启了应用的 **Accessibility 服务**（无障碍服务）。
3.  **启动所有服务**：运行 `INSTALL_AND_START.bat` 启动后端服务。
4.  **开始发号施令**：通过 Android 悬浮窗或 Windows 侧边栏，开始用自然语言控制您的 Galaxy 系统！
